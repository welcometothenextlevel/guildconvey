# Guild Conveyancing Website v3 — Final

Premium dark luxury conveyancing website with full security & privacy messaging.

## Upload to GitHub Pages
1. Upload ALL files to your repo root
2. Settings → Pages → Source: main → / (root)
3. Live in ~2 minutes

## Before going live — update these 4 things:
1. `main.js`: Replace `XXXXXX` with your Formspree form ID (free at formspree.io)
2. All pages: Replace `1300 XXX XXX` with real phone number
3. All pages: Replace `hello@guildconveyancing.com.au` with real email
4. `privacy.html`: Replace `[Address]` with your real business address

## Custom Domain
Add a `CNAME` file to the repo containing `guildconveyancing.com.au`
Set DNS CNAME → `[username].github.io`
